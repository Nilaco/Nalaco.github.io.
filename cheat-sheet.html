CSS Cheat Sheet

Font Properties

font-style:

set font style (ex: 'italic').

----------

font-family:

set font family (ex: 'Times New Roman').

----------

font-weight:

set font weight (ex: 'bold').

----------

font-size:

set font size (ex: '45px').

----------

color:

set font color (ex: 'blue').

----------

text-decoration:

set decoration (ex: 'underline').

Link Properties

:link

set properties for unvisited links

----------

:visited

set properties for visited links

----------

:hover

set properties when input device is hovering over link

----------

:active

set properties when link is being selected

Box Properties

width:
(also min-width/max-width)

assign width value

----------

height:
(also min-height/max-height)

assign height value

----------

overflow:

'hidden' or 'scroll' for content too large to fit in box

----------

border:

box outline (ex: '3px')

----------

margin:

sits outside of border (ex: '15px')

----------

padding:

space between border and contents of box (ex: '8px')

Positioning

Term: Normal Flow

Default behavior. Each block-level element outputs to a new line regardless of how much room is left.

----------

Term: Relative

Moves the element's positioning from where it would be in normal flow by the specified amount (top, bottom, left, right). Does not affect positioning of surrounding elements.

----------

Term: Absolute

Moves the element's positioning in relation to its parent element. Does not affect position of surrounding elements. Moves with user scrolling on page.

----------

Term: Fixed

Element is positioned in relation to the browser window. Does not affect position of surrounding elements. Does not move with user scrolling.

----------

Term: Floating

Takes element out of normal flow, positions to far left or right of parent element. Becomes block-level, and other elements can flow around it.
